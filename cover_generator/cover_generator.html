Voronoi Image Mosaic

Press 'S' to save image | Press 'R' to regenerate

Instructions:

  • Make sure your images folder contains 001.jpg through 102.jpg
  • Run a local server (see instructions below)
  • The sketch will create a Voronoi diagram with random image clips
  • Press 'S' to save the current image
  • Press 'R' to generate a new random arrangement
